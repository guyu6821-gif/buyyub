# BDU Tələbə Köməkçisi

Bakı Dövlət Universiteti tələbələri üçün hesablama və məlumat platforması.

## Xüsusiyyətlər

- ✅ Semestr balı hesablama
- ✅ ÜOMG hesablama
- ✅ 25% İmtahan pulu hesablama
- ✅ Yaş hesablayıcı
- ✅ Akademik lüğət və məlumat bölməsi
- ✅ Sürətli linklər
- ✅ Progressive Web App (PWA) - Offline işləyir
- ✅ Quraşdırma dəstəyi (Install to Home Screen)
- ✅ Tam responsive dizayn

## Texnologiyalar

- HTML5
- CSS3
- Vanilla JavaScript
- Service Worker (PWA)
- Manifest.json

## Quraşdırma

1. Layihəni klonlayın:
```bash
git clone <repository-url>
cd webapp
```

2. Lokal serverdə işlədin (Python ilə):
```bash
python -m http.server 8000
```

Və ya Node.js ilə:
```bash
npx serve .
```

3. Brauzerdə açın: `http://localhost:8000`

## Render.com-da Deploy

1. Render.com hesabınıza daxil olun
2. "New Static Site" yaradın
3. GitHub repository-ni bağlayın
4. Deploy edin

## PWA Xüsusiyyətləri

- 🔄 Avtomatik cache yeniləmə
- 💾 Offline işləmə qabiliyyəti
- 📱 Ana ekrana əlavə oluna bilər
- ⚡ Sürətli yükləmə

## Struktur

```
webapp/
├── index.html          # Ana səhifə
├── styles.css          # CSS dizayn
├── app.js             # JavaScript funksiyaları
├── service-worker.js  # PWA Service Worker
├── manifest.json      # PWA Manifest
├── icon-192.png       # 192x192 ikon
├── icon-512.png       # 512x512 ikon
└── README.md          # Sənədləşdirmə
```
